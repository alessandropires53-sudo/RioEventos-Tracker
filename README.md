RioEventos Tracker

O **RioEventos Tracker** é uma aplicação interativa desenvolvida para facilitar a visualização de eventos culturais, esportivos e gastronômicos na cidade do Rio de Janeiro. 

A ideia surgiu da necessidade de centralizar eventos locais em um mapa intuitivo, permitindo que moradores e turistas explorem a cidade de forma dinâmica.

## Tecnologias Utilizadas
- **HTML5 & CSS3:** Estrutura e estilização moderna com foco em responsividade.
- **JavaScript (ES6+):** Lógica de filtragem, manipulação de dados e integração de mapa.
- **Leaflet.js:** Biblioteca open-source para mapas interativos.
- **Git & GitHub:** Versionamento de código e deploy via GitHub Pages.

##  Funcionalidades
- Exibição de eventos com marcadores personalizados no mapa do Rio de Janeiro.
- Filtro por categorias (Música, Arte, Esporte, Gastronomia).
- Busca em tempo real por nome do evento ou bairro.
- Ajuste automático de zoom do mapa conforme os resultados encontrados.
